Swaks - Swiss Army Knife for SMTP

Ae0lu5
2021-02-08

Swaks是一个功能性，灵活，脚本化，面向事务的SMTP测试工具编写和维护 约翰·杰特莫尔。它可免费使用和许可的GNU GPLv2. 功能包括：

  • SMTP 扩展，包括 TLS、身份验证、管道、代理、PRDR 和 XCLIENT
  • 协议包括 SMTP、ESMTP 和 LMTP
  • 传输，包括 UNIX 域套接字、互联网域套接字（IPv4 和 IPv6）以及生成流程的管道
  • 完全可编写脚本的配置，通过环境变量、配置文件和命令行提供选项规范

官方项目 https://jetmore.org/john/code/swaks/.

下载

最新版本的Swaks是20201014.0 (公告), 它可以下载为 包 或 独立脚本.

有关在多个环境中安装的详细信息，请参阅安装页面。

T还有一个版本页面，其中列出了每个发布版本的 Swaks，并配有更改日志和下载链接。

文档

最新版本的参考文档（包括快速入门示例）以纯文本提供 并 呈现。每个版本中的文档可从页面获得。还有一个偶尔问问题的文件。

通信

欢迎人们提供反馈和有意义的问题，包括如何使用 Swaks。然而，由于Swaks只由一个人作为业余爱好，因此不能保证得到及时的响应。

发布通知

  • Follow @SwaksSMTP关注我们。很少包含非发布内容。
  • Blog. Swaks-specific blog 类别 (可用 RSS)。很少包含非发布内容。

帮助和反馈

  • 问题 - 打开功能请求和 Bug 的问题。
  • 联系作者 - 建议，提示，补丁，反馈，批评总是欢迎。
