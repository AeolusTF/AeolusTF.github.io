R3con1z3r——轻量级Web信息收集工具

Ae0lu5
2021-02-14

R3CON1Z3R

R3con1z3r是一种轻量级Web信息收集工具，具有用python编写的直观功能。它提供了一个强大的环境，在其中可以快速和彻底地进行基于Web的开源智能（OSINT）占用空间。

足迹是道德黑客的第一阶段，它收集了有关目标的所有可能的信息。R3con1z3r是一种被动侦察工具，具有内置功能，包括：HTTP标头标志、跟踪路由、Whois足迹、DNS信息、同一服务器上的站点、Nmap端口扫描仪、反向目标和网页上的超链接。该工具在获得必要的输入后，将生成HTML格式的输出。

安装

r3con1z3r支持Python2和Python3。

模块

r3con1z3r仅依赖于系统并请求python模块。要对窗口进行额外的着色，win_unicode_console安装颜色和颜色

对于Windows上的着色： pip install win_unicode_console colorama

使用

r3con1z3r在系统路径上安装命令行工具，可以使用

使用此工具将生成名为”可以在浏览器上打开”的html，以查看占用结果domain.com-r3con1z3r.html

演示

简单使用r3con1z3r CLI执行OSINT占用google.com

例子

  • 要在所有操作系统（Linux、Windows、Mac OS X、Android e.t.c）上安装，即从克隆存储库中的Python2环境安装：

    pip install r3con1z3r

    python3环境：

    pip3 install r3con1z3r
